### k423-plugin

[yunzai-bot v3](https://github.com/Le-niao/Yunzai-Bot) 插件，想到啥做啥

### 目前的功能

| 功能         | 命令                 |
| ------------ | -------------------- |
| 网页预览     | `#screenshot:<链接>` |
| 卢浮宫生成器 | `#olk<图片>`         |

### 说明

1.  卢浮宫生成器需要安装依赖 `axios` `btoa` `canvas`。其中`canvas`的安装比较麻烦，并且极有可能需要降级 node 版本，而降级 node 版本会带来不可预计的影响，谨慎操作。
